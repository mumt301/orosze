An online Theremin instrument

The theremin is an electronic musical instrument invented by Leon Theremin in the 1920s.

This web app uses pizzicato.js to implement a "virtual" Theremin in to your browser.

The pizzicato.js library (and other similar ones) use the Web Audio API to play sounds on the browser.

The task of Assignment 7 was to implement some upgrades for this virtual instrument which could improve its playability.
